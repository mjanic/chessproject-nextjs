This is a [Next.js](https://nextjs.org/) web app to help chess players improve their openings.

## Preview the app

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
